A highly interactive, physics-driven web application that simulates anime-inspired ocular abilities (Dojutsu) and chakra manipulation. Built entirely with modular Vanilla JavaScript, HTML5 Canvas, and modern CSS3, this project features over 25 unique visual states, kinetic particle systems, and dynamic browser-synthesized audio.

---

## ✨ Features

The engine is heavily armed with interactive modules, divided into four core categories:

### 🎯 Core & UI Systems
* **Modular Architecture:** Clean ES6 module separation for physics, audio, memory, and rendering.
* **Chakra Meter:** A live resource management UI that drains during high-intensity jutsu.
* **Heads-Up Display (HUD):** Slide-out system terminal (`I`) and active systems monitor (`A`).
* **State Memory:** Uses `localStorage` to automatically save and restore your exact visual configuration.
* **Screenshot Engine:** Instantly capture and download high-resolution canvas states.
* **Haptic Feedback:** Physical device vibration for ultimate combos and overdrive states (on supported devices).

### ⚔️ Combat & Tactical Physics
* **Spatial Vortex:** A gravitational singularity that pulls ambient particles to the cursor.
* **Defensive Aura / Ribcage Shield:** Kinetic repulsors that violently push particles away from the center.
* **Chakra Shockwave:** An explosive blast of kinetic force.
* **Amaterasu (Black Fire):** Cursor-tracking black flames that leave permanent scorch marks on the canvas.
* **Ultimate Jutsu Combo:** A hidden sequential keybind (`↑ ↓ ← → Space`) that triggers a massive visual overload.

### 🌌 Reality & Time Manipulation
* **Time Dilation & Time Stop:** Slow particle physics to a crawl, or freeze reality completely.
* **Overdrive Pulse:** Triples physics speed while rhythmically pulsing the UI heartbeat.
* **Phase Shift (Kamui) & Limbo:** Ethereal visual states with opacity drops and peripheral phantom shadows.
* **Genjutsu Inversions:** Flip gravity, invert color spectrums, or plunge the UI into a crimson Tsukuyomi realm.

### 👁️ Optical Visual States
Toggleable eye overlays including:
* **Byakugan Veins** & **Tenseigan Aura**
* **Jougan Lightning** & **Blood Tears**
* **Rasengan Distortion** (Cursor tracking anomaly)
* **Kyuubi Cloak** (Boiling bottom-screen chakra)
* **Chidori Sparks** (Velocity-based cursor lightning)

## 🚀 Installation & Usage

Because this project utilizes **ES6 Modules** (`import`/`export`), it cannot be run by simply double-clicking the `index.html` file due to strict browser CORS policies. You must run it through a local web server.
